# TailwindCSS Coffee Style by huynhyenchi

## Requirements

* **NodeJS v20.14.0**
* **npm 10.7.0**
* **yarn 1.22.22**

Clone project and run test

...
$ git clone https://github.com/huynhyenchi/tailwindcss-coffee-style.git<br/>
$ cd TailwindCSS-Coffee-Style<br/>
$ yarn build:css<br/>
$ yarn start:dev
